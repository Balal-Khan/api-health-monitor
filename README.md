# api-health-monitor

A simple Python command-line tool that checks the health of an HTTP API endpoint, designed to simulate basic service monitoring used in backend and SRE environments.

# Features
-Checks API availability via HTTP status codes
-Supports configurable retries and timeouts
-Handling of network failures
-Fully testable Python package using pytest

# Technologies Used
-Python 3.12
-requests
-pytest
-argparse
